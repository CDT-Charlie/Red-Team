# WinPrivScan

## Author
Adam Belovsky

## Description
WinPrivScan is a read-only Windows privilege escalation enumeration tool. It is designed to help quickly review common local privilege escalation opportunities on Windows systems without modifying the system.

## Features
- System information gathering
- User privilege enumeration
- AlwaysInstallElevated detection
- Unquoted service path detection
- PATH review
- Scheduled task enumeration
- Service overview

## Usage
```bash
python winprivscan.py
